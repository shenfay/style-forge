# Style Forge - 场景化设计配置器

## 产品定位

**场景化设计配置器** - 不只是选颜色，而是在真实UI组件中实时预览效果，一键导出可用配置。

### 核心差异点

| 现有配色网站 | Style Forge |
|-------------|-------------|
| 只提供色板组合 | 真实UI组件预览 |
| 抽象色块展示 | 手机/PC模拟器效果 |
| 无导出功能 | Tailwind Config + AI提示词 |
| 单一维度 | 8维配置矩阵 |

### 目标用户

1. **前端开发者** - 快速生成项目设计系统
2. **UI设计师** - 探索配色方案，验证可访问性
3. **独立开发者** - 一键导出AI提示词给大模型生成代码
4. **产品经理** - 快速原型验证

## 设计系统规范

### 8维配置矩阵

| 维度 | 选项 | 说明 | 影响范围 |
|------|------|------|---------|
| **页面底色** | 纯白/#FFFFFF, 暖沙/#F9F8F4, 淡黄/#FFFDF5, 淡绿/#F5FAF5 | 决定整体视觉温度 | 全局背景 |
| **主题色** | 森林绿/#2E7D32, 草绿/#5F7D2E, 青绿/#2E7D67, 纯黑/#000000 | 品牌识别与交互焦点 | 按钮、标题、高亮 |
| **圆角大小** | 小/8px, 中/16px, 大/24px | 风格基调 | 按钮、卡片、标签 |
| **卡片样式** | 边框/shadow, 阴影/纯阴影, 无装饰/borderless | 容器视觉表现 | 内容区块 |
| **标题栏** | 纯白+下划线, 毛玻璃/frosted, 彩色底色 | 空间感和层级 | 导航区 |
| **切换器** | 下划线/underline, 药丸/pill, 胶囊/capsule | 交互亲密度 | 选项卡组件 |
| **按钮** | 渐变/gradient, 纯色/solid, 线框/wireframe | 行动召唤力度 | 操作按钮 |
| **标签** | 圆角底色/rounded, 纯文字/text-only | 信息密度 | 徽章组件 |

### 配置组合数

理论组合: `4×4×3×3×3×3×3×2 = 7,776` 种风格方案

## 附录

### 风险色定义（固定）

```typescript
const riskColors = {
  danger: { bg: '#FEF2F2', text: '#DC2626', border: '#FECACA' },
  warning: { bg: '#FFFBEB', text: '#D97706', border: '#FDE68A' },
  safe: { bg: '#F0FDF4', text: '#16A34A', border: '#BBF7D0' }
}
```

这些颜色不参与配置，作为语义色固定使用。
