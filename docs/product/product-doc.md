# Style Forge - 场景化设计配置器

## 产品定位

**场景化设计配置器** - 不只是选颜色，而是在真实UI组件中实时预览效果，一键导出可用配置。

### 核心差异点

| 现有配色网站 | Style Forge |
|-------------|-------------|
| 只提供色板组合 | 真实UI组件预览 |
| 抽象色块展示 | 手机/PC模拟器效果 |
| 无导出功能 | Tailwind Config + AI提示词 |
| 单一维度 | 8维配置矩阵 |

### 目标用户

1. **前端开发者** - 快速生成项目设计系统
2. **UI设计师** - 探索配色方案，验证可访问性
3. **独立开发者** - 一键导出AI提示词给大模型生成代码
4. **产品经理** - 快速原型验证

## 设计系统规范

### 20维配置矩阵

#### 色彩系统（10维）

| 维度 | 说明 | 影响范围 |
|------|------|---------|
| **主色** | 品牌核心色 | 按钮、链接、高亮 |
| **辅助色** | 次要交互色 | 次要按钮、标签 |
| **强调色** | 点缀色 | 图标、装饰元素 |
| **页面背景色** | 全局底色 | 页面整体背景 |
| **卡片背景色** | 容器底色 | 卡片、面板 |
| **标题色** | 标题文字色 | 各层级标题 |
| **正文色** | 主要文字色 | 段落内容 |
| **辅助色** | 次要文字色 | 提示文字、时间戳 |
| **成功色** | 正向反馈色 | 成功提示、安全标签 |
| **警告色** | 警示色 | 警告提示 |
| **错误色** | 负向反馈色 | 错误提示、危险操作 |

#### 形状系统（6维）

| 维度 | 选项 | 说明 | 影响范围 |
|------|------|------|---------|
| **圆角大小** | small/8px, medium/16px, large/24px | 风格基调 | 按钮、卡片、标签 |
| **卡片样式** | border/shadow/borderless | 容器视觉表现 | 内容区块 |
| **按钮样式** | gradient/solid/wireframe | 行动召唤力度 | 操作按钮 |
| **标签样式** | rounded/text-only | 信息密度 | 徽章组件 |
| **标题栏样式** | white-underline/frosted-glass/colored-bg | 空间感和层级 | 导航区 |
| **切换器样式** | underline/pill/capsule | 交互亲密度 | 选项卡组件 |

#### 间距系统（4维）

| 维度 | 选项 | 说明 | 影响范围 |
|------|------|------|---------|
| **内边距** | compact/medium/relaxed | 容器内部间距 | 卡片、面板 |
| **卡片间距** | small/medium/large | 卡片之间间距 | 列表、网格 |
| **区块间距** | small/medium/large | 大区块间距 | 页面分区 |
| **元素间距** | compact/medium/relaxed | 小元素间距 | 按钮组、标签组 |

#### 文字排版（5维）

| 维度 | 选项 | 说明 | 影响范围 |
|------|------|------|---------|
| **标题装饰** | left-accent/right-accent/bottom-accent/plain | 标题视觉引导 | 区块标题 |
| **标题大小** | small/medium/large | 标题字号 | 各层级标题 |
| **标题字重** | normal/medium/bold | 标题粗细 | 标题文字 |
| **正文字号** | small/medium/large | 正文字号 | 段落内容 |
| **行高** | compact/medium/relaxed | 文字行间距 | 所有文本 |

### 配置组合数与联动机制

**理论组合**: `10×6×4×5 = 1,200` 种基础配置方案（不含联动）

**配置联动策略**（已实现）：
- 大圆角 + 强阴影 → 增强视觉层次
- 大圆角 + 大内边距 → 更饱满的视觉感受
- 大标题 + 粗装饰线 → 强化视觉引导
- 宽松行高 + 大段落间距 → 提升可读性

### 风险色定义（固定）

```typescript
const riskColors = {
  danger: { bg: '#FEF2F2', text: '#DC2626', border: '#FECACA' },
  warning: { bg: '#FFFBEB', text: '#D97706', border: '#FDE68A' },
  safe: { bg: '#F0FDF4', text: '#16A34A', border: '#BBF7D0' }
}
```

这些颜色不参与配置，作为语义色固定使用。
